# Stack Tecnológico

Este proyecto no tiene misterio, pero sí mucha intención. Hemos buscado rendimiento, facilidad de edición para Beatriz y un diseño que respire su marca personal. Aquí lo que hay bajo el capó:

## Core
- **Next.js 15 (App Router):** La base de todo. Rápido, moderno y con SSR para que Google nos quiera.
- **React 19:** Aprovechando las últimas features y hooks.
- **TypeScript:** Porque nos gusta dormir tranquilos sabiendo que los tipos cuadran.

## Estilos y UI
- **Tailwind CSS v4:** Para maquetar a la velocidad de la luz sin salir del HTML.
- **Framer Motion:** Esas animaciones suaves que le dan el toque "pro" a la web (scroll reveal, transiciones, etc.).
- **Lucide React:** Iconos limpios y ligeros.

## Contenido y Datos
- **Sanity CMS:** El cerebro del contenido. Beatriz puede editar textos, blog y secciones sin tocar una línea de código. Headless y flexible.

## Funcionalidad
- **Formspree:** Para el formulario de contacto. Simple, funciona y nos ahorra montar un backend solo para esto.

## Despliegue
- Preparado para **Vercel** o cualquier hosting moderno que soporte Next.js.
